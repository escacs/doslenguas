**The Ultimate Platform for Conlang Learning & Education**

DosLenguas is a modern, high-performance web application designed for enthusiasts, linguists, and creators of constructed languages. It provides a robust suite of tools to create, share, and master conlangs through an interactive, gamified experience.

---

## 🎨 Premium Experience
- **Glassmorphic UI**: A stunning, modern interface built with the latest design principles.
- **Dynamic Animations**: Smooth page transitions and micro-interactions powered by Framer Motion.
- **Responsive Design**: Learn at home or on the go with a fully mobile-optimized layout.

## 🛠️ Advanced Features

### 🧩 Dynamic Course Editor
Creators can build intricate language paths using a state-of-the-art course editor.
- **Custom Sectioning**: Organize your course into logical modules and units.
- **Dependency Mapping**: *[NEW]* Set explicit unlocking requirements. Make specific sections or activities require other milestones to be met first.
- **Vocabulary Manager**: Maintain a central dictionary for your conlang, complete with IPA pronunciation support.

### 🏃 Interactive Activity Runner
A versatile exercise environment supporting various learning styles:
- **Lessons**: Traditional instructional stages with rich content.
- **Flashcards**: Interactive cards with flip animations to master core vocabulary.
- **Fill-in-the-Blanks**: Targeted orthographic and grammatical practice.
- **Translation Stages**: *[NEW]* Direct translation exercises with multi-answer support for diacritics and variations.

### 🍱 Localization & Orthography
- **Ruby Text Support**: Toggleable phonetic annotations (ruby) to help learners master complex scripts.
- **X-SAMPA to IPA**: Effortless typing of phonetic symbols using standard keyboard characters.

### 🤝 Social & Community
- **Personalized Profiles**: Set custom **Profile Pictures (PFP)** and showcase your achievements.
- **Reviews & Suggestions**: *[NEW]* Leave feedback and contribute to the improvement of shared courses.
- **Course Ratings**: A community-driven rating system to highlight the best content.

---

## 🚀 Tech Stack
- **Frontend**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Backend**: [Node.js](https://nodejs.org/) + [Express](https://expressjs.com/)
- **Styling**: Vanilla [Tailwind CSS](https://tailwindcss.com/)
- **Animations**: [Framer Motion](https://www.framer.com/motion/)
- **Data Persistence**: Local JSON-based storage (perfect for lightweight, server-side persistence).
